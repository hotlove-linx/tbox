# TBox 桌面端软件 - 需求文档

> 版本：v1.1
> 日期：2026-03-17
> 状态：需求细化阶段（已同步原型界面）

---

## 1. 项目概述

### 1.1 项目背景

开发一款桌面端软件商城应用（暂定名 **TBox**），为用户提供软件的浏览、搜索、下载、安装及管理等一站式服务，同时支持用户上传和分享自有软件。

### 1.2 目标用户

- 普通 PC 用户，需要便捷地发现、下载和管理常用软件
- 对软件安全性和来源可靠性有要求的用户
- 独立开发者 / 小团队，需要分发和管理自己开发的软件

## 2. 整体布局与导航结构

### 2.1 布局说明

应用采用经典的 **左侧边栏 + 顶部导航栏 + 右侧内容区** 三栏布局：

- **左侧边栏**：品牌 Logo（TBox + 版本号）、分组导航菜单，包含"发现"（首页、分类浏览）、"管理"（下载管理、安装记录）、"空间"（个人空间）、"系统"（设置）四个分组
- **顶部导航栏**：全局搜索框、下载管理快捷入口（带未读红点）、通知铃铛（带未读红点）、用户信息区（头像 + 昵称 + 邮箱 + 下拉箭头，点击进入个人中心）、窗口控制按钮（最小化 / 最大化 / 关闭）
- **内容区**：根据导航切换展示对应页面内容

### 2.2 导航菜单结构

| 分组 | 菜单项 | 说明 |
|------|--------|------|
| 发现 | 首页 | 推荐 + 分类 + 排行 |
| 发现 | 分类浏览 | 按类别筛选软件 |
| 管理 | 下载管理 | 显示进行中任务数角标 |
| 管理 | 安装记录 | 已安装软件列表 |
| 空间 | 个人空间 | 我的软件 / 上传任务 / 下载记录 |
| 系统 | 设置 | 通用 / 下载 / 通知 / 关于 |

### 2.3 顶部导航栏

- **全局搜索框**：圆角搜索输入框，placeholder 提示"搜索软件、工具、插件..."，支持实时搜索建议
- **下载快捷入口**：图标按钮，有下载任务时显示红点，点击跳转下载管理页
- **通知入口**：铃铛图标，有未读通知时显示红点
- **用户信息区**：显示用户头像、昵称、邮箱，点击跳转个人中心页面
- **窗口控制**：最小化、最大化/还原、关闭按钮

## 3. 功能模块详细设计

### 3.1 用户账户模块

#### 3.1.1 登录 / 注册页

独立全屏页面，居中卡片式布局，顶部展示品牌 Logo（TBox）和 Slogan，通过 Tab 切换登录和注册表单。

#### 3.1.2 邮箱注册

- 用户通过邮箱 + 密码进行注册
- 发送邮箱验证码，验证通过后完成注册
- 密码要求：最少 8 位，包含字母和数字
- 需输入确认密码，两次密码一致才可提交
- 注册时需勾选同意用户协议和隐私政策

#### 3.1.3 邮箱登录

- 支持邮箱 + 密码登录
- 支持"记住我"功能（Token 持久化，有效期 30 天）
- 登录失败 5 次后触发验证码验证或临时锁定（15 分钟）
- 支持"忘记密码"功能，通过邮箱验证码重置密码
- 登录/注册表单底部提供互相跳转链接（"还没有账号？立即注册" / "已有账号？立即登录"）

#### 3.1.4 个人中心

个人中心通过顶部导航栏用户头像进入，页面包含以下区域：

- **头像与基本信息**：显示头像（支持更换）、昵称
- **资料编辑**：编辑昵称，邮箱和注册时间为只读展示
- **修改密码**：输入原密码、新密码、确认新密码
- **账户操作**：退出登录按钮、注销账户入口

---

### 3.2 软件商城模块

#### 3.2.1 首页展示

- **轮播推荐区**：Banner 轮播展示编辑推荐 / 热门软件（3~5 个位置），包含软件名称、简介、下载按钮、装饰图标，底部圆点指示器
- **分类标签栏**：横向排列的分类标签（全部、办公工具、开发工具、影音娱乐、系统工具、安全防护、设计创意、网络通讯），支持点击切换高亮
- **热门推荐网格**：卡片网格展示（自适应列数，最小宽度 200px），每张卡片包含图标、名称、简介（最多2行截断）、评分星级、文件大小。右上角"查看全部"链接
- **下载排行榜**：列表形式展示，每行包含排名序号（前3名高亮）、软件图标、名称、简介、下载次数。右上角"完整榜单"链接

#### 3.2.2 分类浏览页

- 顶部分类标签栏（与首页联动），点击切换当前分类
- 搜索框：在当前分类内搜索
- 筛选工具栏：排序下拉（按热度 / 评分 / 更新时间 / 大小）、价格筛选（全部 / 免费 / 付费）
- 软件卡片网格展示（与首页卡片样式一致）
- 分页加载 / 无限滚动

#### 3.2.3 软件详情页

顶部返回按钮，点击返回上一页。页面分为左侧主区域和右侧信息栏两栏布局：

**头部信息区：**
- 大尺寸软件图标（88x88）、软件名称、开发者名称
- 标签：分类标签、特性标签（如"免费"）
- 核心数据：评分、下载量、文件大小、版本号（横向排列）
- 操作按钮：下载安装（主按钮）、分享（次级按钮）

**左侧主区域：**
- 软件截图：横向滚动展示多张截图预览
- 软件介绍：详细功能描述、特性列表
- 用户评价：评价数量标题、每条评价包含头像、昵称、日期、星级评分、评价文字

**右侧信息栏：**
- 软件信息卡片：版本、大小、开发者、分类、系统要求、更新日期、许可证
- 更新日志卡片：按版本号降序列出，每条包含版本号、日期、更新内容

#### 3.2.4 搜索功能

- 全局搜索框位于顶部导航栏，全局可用
- 实时搜索建议（输入时下拉提示）
- 搜索结果页，支持排序和筛选
- 搜索历史记录（本地存储，可清除）
- 无结果时提供推荐软件

### 3.3 下载管理模块

#### 3.3.1 下载功能

- 点击下载后加入下载队列
- 支持**断点续传**
- 支持**多任务并行下载**（可配置最大并发数，默认 3 个）
- 下载进度实时显示（进度条、百分比、已下载/总大小、速度、剩余时间）
- 下载完成后桌面 / 系统通知提醒
- 顶部导航栏下载图标显示未读红点，提示有进行中的任务

#### 3.3.2 下载列表管理

通过 Tab 标签页切换三种状态视图，每个 Tab 显示对应数量角标：

- **下载中**（Tab）：显示进行中的任务列表，每条包含软件图标、名称版本、进度条、百分比与大小、速度与剩余时间，支持暂停/继续/取消操作按钮。暂停状态下进度条变为警告色
- **已完成**（Tab）：显示已下载文件列表，每条包含软件图标、名称版本、文件大小与完成时间，操作链接：打开文件 / 打开文件夹 / 一键安装 / 删除记录
- **已失败**（Tab）：显示失败任务，支持重试

**批量操作工具栏**：全部暂停、全部继续、清空已完成记录

#### 3.3.3 下载设置

- 自定义默认下载目录（支持浏览选择）
- 设置最大并发下载数（下拉选择：1/2/3/5）
- 设置下载限速（输入框 + MB/s 单位，0 表示不限速）
- 下载完成后自动安装选项（开关）

---

### 3.4 安装记录管理模块

#### 3.4.1 安装记录列表

采用**表格列表**展示所有通过本软件安装的应用，列定义：

| 列名 | 内容 |
|------|------|
| 软件名称 | 图标 + 软件名 + 开发者 |
| 版本 | 当前安装的版本号 |
| 大小 | 安装包文件大小 |
| 安装路径 | 本地安装目录路径 |
| 安装时间 | 安装日期 |
| 状态 | 已安装（绿色） / 可更新（黄色，显示新版本号） / 已卸载（灰色） |
| 操作 | 根据状态显示不同按钮 |

**工具栏功能：**
- 搜索框：按软件名称搜索
- 排序下拉：按安装时间 / 按名称 / 按大小
- 状态筛选下拉：全部状态 / 已安装 / 可更新 / 已卸载

#### 3.4.2 安装操作

- 下载完成后可一键安装（调用系统安装程序）
- 安装过程状态跟踪（安装中 / 安装成功 / 安装失败）
- 安装完成后记录安装信息到本地数据库

#### 3.4.3 软件更新检测

- 定期检查已安装软件是否有新版本
- 有更新时在安装记录列表状态列标记"可更新"并显示目标版本号
- 操作列显示"更新"主按钮，支持一键更新（下载新版本并安装）

#### 3.4.4 卸载管理

- 安装记录操作列提供"卸载"按钮（红色警示样式）
- 调用系统卸载程序执行卸载
- 卸载后更新安装记录状态为"已卸载"
- 可选：清理残留文件

### 3.5 个人空间模块

#### 3.5.1 空间概览

页面顶部展示三张统计卡片：

| 统计卡片 | 展示内容 |
|----------|----------|
| 已上传软件 | 总数，细分展示（X 个已上架 · X 个审核中 · X 个私有） |
| 总下载量 | 累计下载总数 + 本周增量 |
| 空间容量 | 已用 / 总容量（数值 + 进度条） |

#### 3.5.2 Tab 标签页

空间内容通过三个 Tab 标签页组织，每个 Tab 显示数量角标：

**Tab 1 - 我的软件：**

表格列表展示已上传的软件，列定义：

| 列名 | 内容 |
|------|------|
| 软件名称 | 图标 + 名称 |
| 版本 | 当前版本号 |
| 大小 | 文件大小 |
| 可见性 | 公开（蓝色）/ 私有（灰色）/ 链接分享（橙色） |
| 审核状态 | 已通过（绿色）/ 审核中（黄色）/ 已拒绝（红色）/ —（私有不需审核） |
| 下载量 | 下载次数（私有/审核中显示"—"） |
| 上传时间 | 上传日期 |
| 操作 | 根据状态不同：下载 / 编辑 / 更新 / 统计 / 撤回 / 公开 / 分享 / 复制链接 |

工具栏：搜索框 + "上传软件"按钮

**Tab 2 - 上传任务：**

展示软件上传的传输任务列表：

- **上传中**：显示文件名、大小、进度条（渐变紫色）、百分比、速度、剩余时间，支持暂停/取消
- **已暂停**：进度暂停，支持继续/取消
- **已完成**：显示文件名、大小、"上传成功"标签、完成时间，操作链接：查看详情 / 提交审核 / 删除记录

工具栏：新建上传、全部暂停、清空已完成

**Tab 3 - 下载记录：**

展示从个人空间下载自有软件的传输记录：

- **下载中**：显示文件名、大小、"下载中"标签、进度条（绿色）、百分比、速度、剩余时间，支持暂停/取消
- **已完成**：显示文件名、大小、"下载完成"标签、完成时间，操作链接：打开文件 / 打开文件夹 / 一键安装 / 删除记录
- **下载失败**：显示文件名、大小、"下载失败"标签，操作链接：重新下载 / 删除记录

工具栏：全部暂停、全部继续、清空已完成

#### 3.5.3 软件上传（弹窗）

点击"上传软件"或"新建上传"按钮，弹出模态对话框：

- **文件上传区**：拖拽上传区域，支持拖拽文件或点击选择文件，提示支持格式（.exe、.msi、.zip）和大小限制（单个文件最大 2GB）
- **必填信息**：软件名称、版本号、分类（下拉选择）、简介
- **选填信息**：详细描述（多行文本）、软件图标（推荐 256x256 PNG）、软件截图（最多5张）
- **可见性选择**：单选项 — 公开（需审核）/ 私有 / 链接分享
- **底部操作**：取消、提交上传

支持大文件分片上传，断点续传。

#### 3.5.4 审核与上架

- 设置为公开的软件需经过平台审核（安全扫描 + 人工审核）
- 审核状态：待审核 / 审核中 / 已通过 / 已拒绝
- 审核拒绝时给出拒绝原因，支持修改后重新提交
- 审核通过后自动上架至软件商城
- 审核中的软件可撤回

#### 3.5.5 数据统计

- 查看每个软件的下载量趋势（日 / 周 / 月）
- 查看用户评分和评论
- 空间容量使用情况（概览卡片中展示）

### 3.6 系统设置模块

设置页面采用**左侧导航 + 右侧内容**的子布局，左侧为设置分类导航，右侧展示对应设置项。每个设置项以卡片形式组织，卡片内设置行采用"标签说明 — 控件"的左右对齐布局。

#### 3.6.1 通用设置

| 设置项 | 说明 | 控件类型 |
|--------|------|----------|
| 开机自启动 | 系统启动时自动运行 TBox | 开关 |
| 最小化到系统托盘 | 关闭窗口时最小化到托盘而非退出 | 开关 |
| 语言 | 设置界面显示语言 | 下拉选择（简体中文 / English） |
| 主题 | 设置界面主题风格 | 下拉选择（浅色 / 深色 / 跟随系统） |

#### 3.6.2 下载设置

| 设置项 | 说明 | 控件类型 |
|--------|------|----------|
| 默认下载路径 | 软件下载文件的保存位置 | 路径输入框 + 浏览按钮 |
| 最大并发下载数 | 同时下载任务的最大数量 | 下拉选择（1/2/3/5） |
| 下载限速 | 限制下载速度（0 表示不限速） | 数字输入框 + MB/s 单位 |
| 下载完成后自动安装 | 软件下载完成后自动执行安装 | 开关 |

#### 3.6.3 通知设置

| 设置项 | 说明 | 控件类型 |
|--------|------|----------|
| 下载完成通知 | 下载任务完成时推送通知 | 开关 |
| 软件更新提醒 | 已安装软件有新版本时提醒 | 开关 |
| 通知方式 | 选择通知的展示方式 | 下拉选择（系统通知 / 应用内通知 / 两者都有） |

#### 3.6.4 关于

| 设置项 | 说明 | 控件类型 |
|--------|------|----------|
| 当前版本 | 显示版本号和构建信息 | 文本 + 检查更新按钮 |
| 用户协议 | 查看用户协议 | 链接 |
| 隐私政策 | 查看隐私政策 | 链接 |
| 反馈与帮助 | 提交反馈和问题 | 链接 |

## 4. 非功能性需求

### 4.1 性能要求

| 指标 | 要求 |
|------|------|
| 应用启动时间 | ≤ 3 秒（冷启动） |
| 商城首页加载 | ≤ 2 秒 |
| 搜索响应时间 | ≤ 500 毫秒 |
| 下载速度 | 充分利用用户带宽，不低于浏览器下载速度 |
| 上传速度 | 充分利用用户上行带宽，支持分片并行上传 |
| 内存占用 | 空闲状态 ≤ 200MB |

### 4.2 安全要求

- 用户密码加密存储（bcrypt / argon2）
- 通信全程 HTTPS 加密
- Token 安全存储（系统密钥链 / 加密本地存储）
- 下载文件 Hash 校验（SHA-256），确保文件完整性
- 上传文件安全扫描（病毒 / 恶意代码检测）
- 软件来源可信验证（数字签名校验）

### 4.3 兼容性要求

- 操作系统：Windows 10 及以上
- 屏幕分辨率：最低支持 1280 x 720，适配高 DPI 屏幕
- 网络：支持 HTTP 代理设置

### 4.4 可用性要求

- 界面简洁直观，核心操作路径不超过 3 步
- 支持键盘快捷键操作
- 异常情况有明确的错误提示和引导
- 表格列表支持 hover 高亮行
- 操作按钮区分主次样式（主操作高亮色、危险操作红色警示）

## 8. 风险与待确认项

| 编号 | 待确认项 | 说明 |
|------|----------|------|
| 1 | 软件来源 | 软件资源是自建仓库还是聚合第三方来源？ |
| 2 | 付费模式 | 是否存在付费软件？如有，需要集成支付系统 |
| 3 | 软件审核 | 上架软件是否需要安全审核流程？ |
| 4 | 跨平台 | 初期是否仅支持 Windows，后续是否考虑 macOS / Linux？ |
| 5 | 后台管理 | 是否需要同步开发管理后台（软件管理、用户管理等）？ |
| 6 | 自动安装 | 静默安装的兼容性和安全性如何保证？ |
| 7 | 存储方案 | 用户上传的软件文件存储在哪里？（自建对象存储 / 云存储服务如 OSS、S3） |
| 8 | 上传限制 | 用户默认空间大小、单文件大小上限、支持的文件格式是否需要调整？ |
| 9 | 内容审核 | 审核机制是纯人工还是自动安全扫描 + 人工复审？ |
