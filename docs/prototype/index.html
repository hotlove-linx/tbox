软件介绍

Visual Studio Code 是一款由 Microsoft 开发的免费、开源的代码编辑器。它支持语法高亮、智能代码补全（IntelliSense）、代码重构、内置调试器和 Git 集成。

VS Code 拥有丰富的扩展生态系统，支持 JavaScript、TypeScript、Python、Java、C++ 等数百种编程语言。用户可通过扩展市场安装各类插件来增强功能，如主题、代码片段、语言支持、调试工具等。

主要特性：
• IntelliSense 智能补全和语法检查
• 内置 Git 版本控制支持
• 集成终端和调试工具
• 跨平台支持（Windows / macOS / Linux）
• 轻量启动，性能优异
